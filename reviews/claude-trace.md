# Claude Trace

> [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=nrigalle.claude-trace) — визуальная история Claude Code: сессии, чат, затронутые файлы и графики использования в одном месте.

## Что это

VS Code-расширение для просмотра истории работы Claude Code. Оно собирает в одном интерфейсе прошлые сессии, переписку, затронутые файлы и графики использования, поэтому не приходится вручную разбираться в разрозненных логах.

## Зачем

- Быстро возвращаться к прошлым сессиям Claude Code.
- Видеть чат и изменённые файлы рядом.
- Понимать, что агент делал в длинной задаче.
- Анализировать статистику и графики использования.
- Получать нормальный визуальный обзор вместо чтения сырых логов.

## Что особенно хорошо

Расширение выглядит неожиданно цельным и отполированным для проекта с пока небольшим числом установок. Интерфейс красивый и удобный: основные данные действительно собраны в одном месте и читаются сразу.

## Установка

1. Открыть страницу [Claude Trace в VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=nrigalle.claude-trace).
2. Нажать **Install**.
3. Открыть Claude Trace в VS Code и выбрать нужную сессию.

## Как использовать

- Просматривать список сессий Claude Code.
- Открывать переписку конкретной сессии.
- Проверять, какие файлы были затронуты.
- Смотреть графики и статистику использования.

## Ограничения

- Пока не хватает поддержки реальных имён кастомных моделей. Например, при использовании GLM расширение может показывать модель как Opus.
- Полезность истории зависит от того, какие данные Claude Code сохранил локально.
