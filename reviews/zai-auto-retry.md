# zai-auto-retry

> [github.com/megamen32/zai-auto-retry](https://github.com/megamen32/zai-auto-retry) — автоповтор запроса в Z.AI при появлении peak-hours диалога.

## Что это

Узкий, но очень практичный Chrome-экст. Когда Z.AI показывает модалку «Currently in peak hours» и фактически отказывает в ответе, расширение само сохраняет текущий промпт + прикреплённые файлы, перезагружает чат, восстанавливает ввод и шлёт запрос заново. Через несколько секунд пик обычно проходит.

## Зачем

- Не нажимать retry вручную каждый раз.
- Не терять длинный промпт и файлы при пиковой нагрузке.
- Дожимать запросы, которые почти наверняка пройдут на повторе.
- Держать рабочий поток, когда Z.AI нестабилен.

## Как работает

1. Расширение активно только на `https://chat.z.ai/*`.
2. Наблюдает за DOM, ждёт появления peak-hours модалки.
3. Когда модалка появилась:
   - сохраняет текущее поле ввода в Chrome local storage,
   - сохраняет прикреплённые файлы и текущий Authorization header (чтобы перезалить файлы после reload),
   - отменяет упавший запрос, перезагружает страницу,
   - восстанавливает промпт, перезаливает файлы где возможно, жмёт Send.
4. Тоггл `ON` / `OFF` по клику на иконке. Badge на иконке показывает текущее состояние.

## Установка

1. `chrome://extensions/` → Developer mode ON.
2. Load unpacked → папка репо.
3. Открыть `https://chat.z.ai/`.
4. Кликнуть по иконке расширения → `ON`.

## Permissions

- `storage` — хранить retry-стейт между reload.
- `https://chat.z.ai/*` — расширение работает только на Z.AI.

## Приватность

Расширение **не отправляет данные наружу**. Всё временное хранится локально в Chrome storage и стирается после инициализации retry. Для поддержки перезалива файлов в Z.AI временно сохраняется содержимое файлов и auth-header — использовать только в своём доверенном профиле браузера.

## Ограничения

- Работает только с Z.AI chat (`chat.z.ai`). Другие peak-hours диалоги (ChatGPT, Claude и т.п.) — мимо.
- Перезагрузка страницы — заметная пауза; для очень длинного контекста часть истории может не восстановиться, если Z.AI не сохранил её на сервере.
- Зависит от текущего UI Z.AI: при смене разметки модалки нужно обновить селектор наблюдателя.
