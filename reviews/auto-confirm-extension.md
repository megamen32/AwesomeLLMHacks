# auto-confirm-extension

> [github.com/megamen32/auto-confirm-extension](https://github.com/megamen32/auto-confirm-extension) — автоклик по диалогам подтверждения в ChatGPT (Custom GPT actions).

## Что это

Маленький Chrome-расширение, форк [Auto Confirm for ChatGPT](https://chromewebstore.google.com/detail/auto-confirm-for-chatgpt/dfiponmapmachnpedjilhecigmmggffk). Само находит и кликает кнопки подтверждения в ChatGPT-диалогах, поддерживает несколько языков через конфиг селекторов.

## Зачем

- Убрать ручную рутину в длинных агентных сессиях, где Custom GPT каждые пару шагов просит подтвердить одинаковый action.
- Не сидеть у экрана ради каждого клика.
- Ускорить отладку и автоматизацию Custom GPT.

## Как работает

- Расширение работает только на страницах ChatGPT.
- Подбирает кнопку `Confirm` через набор селекторов в `CONFIG.SELECTORS` (мультиязычные варианты).
- Если находит — кликает. Если нет — пропускает.
- Включается/выключается тогглом на иконке расширения.

## Установка

1. `chrome://extensions/` → Developer mode ON.
2. Load unpacked → выбрать папку репо.
3. На странице ChatGPT кликнуть по иконке расширения, чтобы включить.

## Отладка

- `CONFIG.DEBUG = true` в `content.js` — режим подробного логирования селекторов в DevTools console.
- При смене UI ChatGPT обновлять значения в `CONFIG.SELECTORS`.

## Ограничения

- Это **экспериментальная** утилита для отладки. Использовать аккуратно: автоподтверждение опасно для действий вроде отправки денег, удаления данных, публикации, выдачи секретов.
- При смене разметки ChatGPT селекторы могут отставать — нужно обновлять вручную.
- Не покрывает диалоги вне ChatGPT (Codex CLI, API и т.п.).
