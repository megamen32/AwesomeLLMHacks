# GPT-AI-Markdown-LaTeX-Copier

> [github.com/Wavesflow/GPT-AI-Markdown-LaTeX-Copier](https://github.com/Wavesflow/GPT-AI-Markdown-LaTeX-Copier) — Tampermonkey-скрипт для копирования ответов LLM в чистый Markdown с сохранением LaTeX.

## Что это

UserScript, который перехватывает `Ctrl+C` на поддерживаемых AI-сайтах и кладёт в буфер обмена **чистый Markdown** вместо тяжёлого HTML, который отдаёт стандартное копирование. Дополнительно старается сохранить LaTeX-формулы в нормальном синтаксисе, пригодном для Obsidian / Typora / VS Code / Notion.

## Зачем

- Копировать ответы в заметки / README / Obsidian без лишнего HTML-мусора и невидимых span-ов.
- Сохранять структуру Markdown: код-блоки, списки, таблицы, ссылки, заголовки.
- Не чинить руками формулы после каждого копирования — LaTeX переносится корректно.
- Работать единообразно на разных LLM-чатах.

## Поддерживаемые сайты

- `chatgpt.com`, `chat.openai.com`
- `claude.ai`
- `deepseek.com`
- `tongyi.aliyun.com` (Qwen)
- `chat.qwen.ai`
- `moonshot.cn`

## Установка

1. Поставить Tampermonkey.
2. Создать новый скрипт → вставить код из репо.
3. Сохранить, **перезапустить браузер** (важно).
4. На поддерживаемом сайте выделить ответ → `Ctrl+C`. В буфер попадёт чистый Markdown.

## Что умеет сохранять

- Блоки кода с правильным языком (```python, ```js и т.п.).
- Списки (нумерованные и маркированные).
- Таблицы.
- Ссылки.
- LaTeX-формулы (`$...$` / `$$...$$`).

## Ограничения

- **Перезапуск браузера** после установки — обязателен, иначе хуки могут не подцепиться.
- Поддерживаются только сайты из списка. Новые сервисы (Grok, AI Studio, Mistral Web) нужно добавлять в `@match` скрипта.
- Скрипт читает выделение и конвертирует его. Если выделение странное (часть таблицы, скрытый текст) — результат может быть неполным.
- Не пытается рендерить Markdown-обратно: это **копировалка**, не просмотрщик.
