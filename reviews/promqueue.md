# promqueue

> [github.com/megamen32/promqueue](https://github.com/megamen32/promqueue) — очередь и bulk-send промптов для z.ai, ChatGPT, Claude, DeepSeek, Qwen, Gemini, AI Studio, Copilot.

## Что это

Форк оригинального [Chat Queue for ChatGPT](https://chromewebstore.google.com/detail/chat-queue-for-chatgpt-cl/oklddkmldemkikacoecmaanmakkplggj), расширенный поддержкой z.ai, DeepSeek, Qwen и др. Кладёшь промпты в очередь, пока текущий ответ ещё генерируется — расширение само отправит следующий, как только чат освободится.

## Зачем

- Не ждать руками каждый ответ перед отправкой следующего запроса.
- Прогонять серии однотипных промптов: переписывание, генерация вариантов, пакетная обработка списков.
- Использовать один queue-интерфейс для разных LLM-сервисов, переключаясь per-site.
- Превращать веб-чаты в полуавтоматический batch-конвейер.

## Установка

1. Клонировать репо или скачать ZIP.
2. `chrome://extensions/` → Developer mode ON.
3. Load unpacked → выбрать папку расширения.
4. Открыть любой поддерживаемый чат → иконка PrompQueue или `Ctrl/Cmd+Shift+Q`.

## Использование

1. Включить расширение на нужной платформе.
2. Открыть queue-панель (иконка или шорткат).
3. Набить промптов в очередь.
4. Auto-send: промпты уходят по одному, как только текущий ответ завершается.
5. Per-site настройки — отдельно для каждой платформы.


## Ограничения

- Это **форк** оригинального расширения для ChatGPT — поддержка новых платформ держится на community-контрибьютах и может отставать при изменениях UI.
- Промпты в очереди не параметризованы — каждый шлётся как есть. Для вариаций нужны шаблоны на стороне пользователя.
- Нет retry-логики при ошибках конкретных платформ (если сервис упал, промпт останется в очереди).
